# Security Policy

## Supported Versions

The only supported version of this application is the version deployed to [movies.wowellworld.com](https://movies.wowellworld.com) from `main`.

## Reporting a Vulnerability

Please do not add any issues for security vulnerabilities. 

Report any security vulnerabilities directly to [jordan@jordanwelch.com](mailto:jordan@jordanwelch.com).
